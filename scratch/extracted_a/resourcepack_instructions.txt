             SECRET NEIGHBOR MINIGAME - RESOURCE PACK GUIDE
========================================================================

Folder ini berisi struktur dasar Minecraft Resource Pack yang dikonfigurasi
untuk plugin Secret Neighbor Anda menggunakan Custom Model Data.

------------------------------------------------------------------------

------------------------------------------------------------------------
2. CARA MENAMBAHKAN MODEL 3D & TEKSTUR ANDA
------------------------------------------------------------------------

A. File Model 3D (Blockbench .json):
   - Jika Anda memiliki model 3D Blockbench untuk Kardus, Bat, Kamera, Kunci, dll.
   - Ekspor model tersebut dari Blockbench sebagai ".json" model item Minecraft.
   - Pindahkan file model tersebut ke folder:
     `assets/secretneighbor/models/item/`
   - Beri nama file sesuai dengan nama pada tabel di atas (misalnya `cardboard_box.json` untuk Kardus).

B. File Tekstur (.png):
   - Pindahkan gambar tekstur .png dari model Anda ke folder:
     `assets/secretneighbor/textures/item/`
   - Pastikan di dalam file model .json Anda, path teksturnya mengarah ke:
     `secretneighbor:item/nama_file_tekstur`

------------------------------------------------------------------------
3. CARA MEMASANG DI MINECRAFT
------------------------------------------------------------------------

1. Zip seluruh isi dari folder `resourcepack/` (bukan folder luarnya, melainkan
   file `pack.mcmeta` dan folder `assets` berada di root file ZIP).
2. Buka Minecraft, masuk ke Options -> Resource Packs -> Open Pack Folder.
3. Taruh file ZIP hasil kompresi tersebut ke dalam folder tersebut.
4. Aktifkan resource pack tersebut di dalam game.

Sekarang, semua item kustom di plugin Secret Neighbor akan menampilkan model 3D
dan tekstur kustom Anda secara otomatis!
